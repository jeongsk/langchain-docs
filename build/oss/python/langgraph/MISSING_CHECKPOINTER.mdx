checkpointer를 제공하지 않고 LangGraph의 내장 지속성 기능을 사용하려고 시도하고 있습니다.

이는 [`StateGraph`](https://langchain-ai.github.io/langgraph/reference/graphs/#langgraph.graph.StateGraph)의 `compile()` 메서드 또는 [`entrypoint`](https://langchain-ai.github.io/langgraph/reference/func/#langgraph.func.entrypoint)에 `checkpointer`가 누락된 경우 발생합니다.

## 문제 해결

다음 사항이 이 오류를 해결하는 데 도움이 될 수 있습니다:

-   checkpointer를 초기화하고 [`StateGraph`](https://langchain-ai.github.io/langgraph/reference/graphs/#langgraph.graph.StateGraph)의 `compile()` 메서드 또는 [`entrypoint`](https://langchain-ai.github.io/langgraph/reference/func/#langgraph.func.entrypoint)에 전달하세요.

```python
from langgraph.checkpoint.memory import InMemorySaver
checkpointer = InMemorySaver()

# Graph API
graph = StateGraph(...).compile(checkpointer=checkpointer)

# Functional API
@entrypoint(checkpointer=checkpointer)
def workflow(messages: list[str]) -> str:
    ...
```





-   LangGraph API를 사용하면 checkpointer를 수동으로 구현하거나 구성할 필요가 없습니다. API가 모든 지속성 인프라를 자동으로 처리합니다.
