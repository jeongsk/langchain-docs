이 페이지는 VertexAI [채팅 모델](/oss/python/langchain/models)을 시작하기 위한 빠른 개요를 제공합니다. ChatVertexAI의 모든 기능과 구성에 대한 자세한 문서는 [API 참조](https://python.langchain.com/api_reference/google_vertexai/chat_models/langchain_google_vertexai.chat_models.ChatVertexAI.html)를 참조하세요.

ChatVertexAI는 `gemini-2.5-pro`, `gemini-2.5-flash` 등과 같이 Google Cloud에서 사용 가능한 모든 기반 모델을 노출합니다. 사용 가능한 모델의 전체 및 최신 목록은 [VertexAI 문서](https://cloud.google.com/vertex-ai/generative-ai/docs/models)를 참조하세요.

<Info>
**Google Cloud VertexAI vs Google PaLM**

Google Cloud VertexAI 통합은 [Google PaLM 통합](/oss/python/integrations/chat/google_generative_ai/)과 별개입니다. Google은 GCP를 통해 PaLM의 엔터프라이즈 버전을 제공하기로 선택했으며, 이는 그곳에서 제공되는 모델을 지원합니다.

</Info>

## 설정

VertexAI 모델에 접근하려면 Google Cloud Platform 계정을 생성하고, 인증 정보를 설정하고, `langchain-google-vertexai` 통합 패키지를 설치해야 합니다.

### 인증 정보

통합을 사용하려면 다음 중 하나를 수행해야 합니다:

- 환경에 대해 구성된 인증 정보 보유 (gcloud, workload identity 등...)
- 서비스 계정 JSON 파일 경로를 GOOGLE_APPLICATION_CREDENTIALS 환경 변수로 저장

이 코드베이스는 위에서 언급한 애플리케이션 인증 정보 변수를 먼저 찾고, 그 다음 시스템 수준 인증을 찾는 `google.auth` 라이브러리를 사용합니다.

자세한 정보는 다음을 참조하세요:

- [cloud.google.com/docs/authentication/application-default-credentials#GAC](https://cloud.google.com/docs/authentication/application-default-credentials#GAC)
- [googleapis.dev/python/google-auth/latest/reference/google.auth.html#module-google.auth](https://googleapis.dev/python/google-auth/latest/reference/google.auth.html#module-google.auth)

모델 호출의 자동 추적을 활성화하려면 [LangSmith](https://docs.smith.langchain.com/) API 키를 설정하세요:

```python
# os.environ["LANGSMITH_API_KEY"] = getpass.getpass("Enter your LangSmith API key: ")

```

## 인스턴스화

이제 모델 객체를 인스턴스화하고 채팅 완성을 생성할 수 있습니다:

```python
from langchain_google_vertexai import ChatVertexAI

llm = ChatVertexAI(
    model="gemini-2.5-flash",
    temperature=0,
    max_tokens=None,
    max_retries=6,
    stop=None,
    # other params...
)
```

## 내장 도구

Gemini는 서버 측에서 실행되는 다양한 도구를 지원합니다.

### Google 검색

<Info>
    **`langchain-google-vertexai>=2.0.11` 필요**
</Info>

Gemini는 Google 검색을 실행하고 결과를 사용하여 [응답을 그라운딩](https://ai.google.dev/gemini-api/docs/grounding)할 수 있습니다:

```python
from langchain_google_vertexai import ChatVertexAI

llm = ChatVertexAI(model="gemini-2.5-flash").bind_tools([{"google_search": {}}])

response = llm.invoke("What is today's news?")
```

### 코드 실행

<Info>
    **`langchain-google-vertexai>=2.0.25` 필요**
</Info>

Gemini는 [Python 코드를 생성하고 실행](https://ai.google.dev/gemini-api/docs/code-execution)할 수 있습니다:

```python
from langchain_google_vertexai import ChatVertexAI

llm = ChatVertexAI(model="gemini-2.5-flash").bind_tools([{"code_execution": {}}])

response = llm.invoke("What is 3^3?")
```
