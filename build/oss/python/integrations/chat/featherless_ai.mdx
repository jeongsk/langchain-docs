## 설정

Featherless AI 모델에 접근하려면 Featherless AI 계정을 생성하고, API 키를 받고, `langchain-featherless-ai` 통합 패키지를 설치해야 합니다.

### 인증 정보

[featherless.ai/](https://featherless.ai/)로 이동하여 FeatherlessAI에 가입하고 API 키를 생성하세요. 완료한 후 FEATHERLESSAI_API_KEY 환경 변수를 설정하세요:

```python
import getpass
import os

if not os.getenv("FEATHERLESSAI_API_KEY"):
    os.environ["FEATHERLESSAI_API_KEY"] = getpass.getpass(
        "Enter your FeatherlessAI API key: "
    )
```

모델 호출의 자동 추적을 사용하려면 아래 주석을 해제하여 [LangSmith](https://docs.smith.langchain.com/) API 키를 설정할 수도 있습니다:

```python
# os.environ["LANGSMITH_TRACING"] = "true"
# os.environ["LANGSMITH_API_KEY"] = getpass.getpass("Enter your LangSmith API key: ")
```

## 인스턴스화

이제 모델 객체를 인스턴스화하고 채팅 완성을 생성할 수 있습니다:

```python
from langchain_featherless_ai import ChatFeatherlessAi

llm = ChatFeatherlessAi(
    model="featherless-ai/Qwerky-72B",
    temperature=0.9,
    max_tokens=None,
    timeout=None,
)
```
