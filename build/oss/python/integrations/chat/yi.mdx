이 가이드는 Yi [채팅 모델](/oss/python/langchain/models)을 시작하는 데 도움을 줍니다. 모든 ChatYi 기능과 구성에 대한 자세한 문서는 [API 레퍼런스](https://python.langchain.com/api_reference/lanchain_community/chat_models/lanchain_community.chat_models.yi.ChatYi.html)를 참조하세요.

AI 2.0의 최전선에 있는 글로벌 기업인 [01.AI](https://www.lingyiwanwu.com/en)는 Dr. Kai-Fu Lee가 설립했습니다. 6B에서 수천억 개의 매개변수에 이르는 Yi 시리즈를 포함한 최첨단 대규모 언어 모델을 제공합니다. 01.AI는 또한 멀티모달 모델, 오픈 API 플랫폼, Yi-34B/9B/6B 및 Yi-VL과 같은 오픈 소스 옵션을 제공합니다.

## 설정

ChatYi 모델에 액세스하려면 01.AI 계정을 생성하고 API 키를 받은 다음 `langchain_community` 통합 패키지를 설치해야 합니다.

### 자격 증명

01.AI에 가입하고 API 키를 생성하려면 [01.AI](https://platform.01.ai)로 이동하세요. 완료한 후 `YI_API_KEY` 환경 변수를 설정하세요:

```python
import getpass
import os

if "YI_API_KEY" not in os.environ:
    os.environ["YI_API_KEY"] = getpass.getpass("Enter your Yi API key: ")
```

모델 호출의 자동 추적을 활성화하려면 [LangSmith](https://docs.smith.langchain.com/) API 키를 설정하세요:

```python
# os.environ["LANGSMITH_API_KEY"] = getpass.getpass("Enter your LangSmith API key: ")
# os.environ["LANGSMITH_TRACING"] = "true"
```
