이 노트북은 복사하여 붙여넣은 내용에서 문서 객체를 로드하는 방법을 다룹니다. 이 경우 DocumentLoader를 사용할 필요도 없이 Document를 직접 구성할 수 있습니다.

```python
from langchain_core.documents import Document
```

```python
text = "..... 여기에 복사하여 붙여넣은 텍스트를 입력하세요......"
```

```python
doc = Document(page_content=text)
```

## 메타데이터

이 텍스트 조각을 어디서 가져왔는지에 대한 메타데이터를 추가하려면 metadata 키를 사용하여 쉽게 추가할 수 있습니다.

```python
metadata = {"source": "internet", "date": "Friday"}
```

```python
doc = Document(page_content=text, metadata=metadata)
```
