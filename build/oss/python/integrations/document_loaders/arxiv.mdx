[arXiv](https://arxiv.org/)는 물리학, 수학, 컴퓨터 과학, 양적 생물학, 양적 금융, 통계, 전기 공학 및 시스템 과학, 경제학 분야의 200만 편의 학술 논문을 위한 오픈 액세스 아카이브입니다.

## 설정

Arxiv 문서 로더에 액세스하려면 `arxiv`, `PyMuPDF` 및 `langchain-community` 통합 패키지를 설치해야 합니다. PyMuPDF는 arxiv.org 사이트에서 다운로드한 PDF 파일을 텍스트 형식으로 변환합니다.

```python
%pip install -qU langchain-community arxiv pymupdf
```

## 인스턴스화

이제 모델 객체를 인스턴스화하고 문서를 로드할 수 있습니다:

```python
from langchain_community.document_loaders import ArxivLoader

# Supports all arguments of `ArxivAPIWrapper`
loader = ArxivLoader(
    query="reasoning",
    load_max_docs=2,
    # doc_content_chars_max=1000,
    # load_all_available_meta=False,
    # ...
)
```

## 로드

`.load()`를 사용하여 모든 Document를 메모리에 동기적으로 로드하며, 각 arxiv 논문당 하나의 Document가 생성됩니다.

reasoning에 대한 논문을 검색하는 `ArxivLoader` 사용의 기본 예제를 살펴보겠습니다:

```python
docs = loader.load()
docs[0]
```

## 지연 로드

많은 수의 Document를 로드하고 로드된 모든 Document의 하위 집합에 대해 다운스트림 작업을 수행할 수 있는 경우, 메모리 사용량을 최소화하기 위해 한 번에 하나씩 Document를 지연 로드할 수 있습니다:

```python
docs = []

for doc in loader.lazy_load():
    docs.append(doc)

    if len(docs) >= 10:
        # do some paged operation, e.g.
        # index.upsert(doc)

        docs = []
```

이 예제에서는 한 번에 10개 이상의 Document를 메모리에 로드하지 않습니다.

## 논문 요약을 문서로 사용
