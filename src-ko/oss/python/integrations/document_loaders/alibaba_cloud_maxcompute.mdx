>[Alibaba Cloud MaxCompute](https://www.alibabacloud.com/product/maxcompute) (이전 ODPS)는 대규모 데이터 웨어하우징을 위한 범용 완전 관리형 다중 테넌시 데이터 처리 플랫폼입니다. MaxCompute는 다양한 데이터 가져오기 솔루션과 분산 컴퓨팅 모델을 지원하여 사용자가 대규모 데이터셋을 효과적으로 쿼리하고 생산 비용을 절감하며 데이터 보안을 보장할 수 있습니다.

`MaxComputeLoader`를 사용하면 MaxCompute SQL 쿼리를 실행하고 결과를 행당 하나의 문서로 로드할 수 있습니다.

## Basic Usage

로더를 인스턴스화하려면 실행할 SQL 쿼리, MaxCompute 엔드포인트 및 프로젝트 이름, 그리고 액세스 ID와 시크릿 액세스 키가 필요합니다. 액세스 ID와 시크릿 액세스 키는 `access_id` 및 `secret_access_key` 매개변수를 통해 직접 전달하거나 환경 변수 `MAX_COMPUTE_ACCESS_ID` 및 `MAX_COMPUTE_SECRET_ACCESS_KEY`로 설정할 수 있습니다.

```python
from langchain_community.document_loaders import MaxComputeLoader
```

```python
base_query = """
SELECT *
FROM (
    SELECT 1 AS id, 'content1' AS content, 'meta_info1' AS meta_info
    UNION ALL
    SELECT 2 AS id, 'content2' AS content, 'meta_info2' AS meta_info
    UNION ALL
    SELECT 3 AS id, 'content3' AS content, 'meta_info3' AS meta_info
) mydata;
"""
```

```python
endpoint = "<ENDPOINT>"
project = "<PROJECT>"
ACCESS_ID = "<ACCESS ID>"
SECRET_ACCESS_KEY = "<SECRET ACCESS KEY>"
```

```python
loader = MaxComputeLoader.from_params(
    base_query,
    endpoint,
    project,
    access_id=ACCESS_ID,
    secret_access_key=SECRET_ACCESS_KEY,
)
data = loader.load()
```

```python
print(data)
```


## Specifying Which Columns are Content vs Metadata

`page_content_columns` 및 `metadata_columns` 매개변수를 사용하여 어떤 열이 Document의 내용으로 로드되고 어떤 열이 메타데이터로 로드될지 구성할 수 있습니다.

```python
loader = MaxComputeLoader.from_params(
    base_query,
    endpoint,
    project,
    page_content_columns=["content"],  # Specify Document page content
    metadata_columns=["id", "meta_info"],  # Specify Document metadata
    access_id=ACCESS_ID,
    secret_access_key=SECRET_ACCESS_KEY,
)
data = loader.load()
```

```python
print(data[0].page_content)
```

```output
content: content1
```

```python
print(data[0].metadata)
```

```output
{'id': 1, 'meta_info': 'meta_info1'}
```
